# 🕉️ Ananta Gita

Ananta Gita is an Android application built with **Kotlin** and **Jetpack Compose** that provides a simple, modern, and offline way to read the **Bhagavad Gita**. The app also includes interactive quizzes to help users test and improve their understanding of the Gita.

## ✨ Features

- 📖 Complete Bhagavad Gita (18 Chapters)
- 🌐 Offline Reading
- 📚 Chapter-wise Navigation
- 🔍 Search Chapters
- 📝 Interactive Gita Quiz
- 📊 Quiz Score & Progress
- 🎨 Material 3 Design
- 🌙 Light & Dark Theme
- ⚡ Fast & Lightweight
- 📱 Responsive UI
- 🚫 No Internet Required

## 🛠️ Tech Stack

- Kotlin
- Jetpack Compose
- Material 3
- MVVM Architecture
- Navigation Compose
- Coroutines
